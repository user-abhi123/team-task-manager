# Team Task Manager

Full-stack Team Task Manager web application built using MERN Stack.

## Features

- Login / Register
- Admin & Member Roles
- Punch In / Punch Out Timer
- Project Management
- Create Project
- Delete Project
- Task Assignment
- Task Status Update
- Todo / In Progress / Done
- Dashboard

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js
- MongoDB
