# Team Task Manager

A full-stack MERN project management web app where users can:

- Create projects
- Assign tasks
- Track task progress
- Manage attendance
- Role-based authentication (Admin/Member)

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js
- MongoDB

## Features

- JWT Authentication
- Project Management
- Task Assignment
- Attendance System
- Dashboard Analytics
- Role-Based Access Control

## Deployment

Frontend: Railway  
Backend: Railway  
Database: MongoDB Atlas / Railway MongoDB
